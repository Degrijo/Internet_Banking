This project is my first web project. It builds on the Django technology in backend and Bootstrap technology in frontend. Project has 6 applications: main, map, news, mytransactions, registration and rate. ATM (it's just a name, because it's more like with internet-banking) use sqlite3 database and 2 models, 2 form in it. The first model is overrided standart User model (was added IntegerField for cash and some methods). The second is article model for news. Form is based on the user model. There are 3 APIs in this project: 2 Belarusbank (location of the ATMs and rate in reqional towns) and 1 Yandex (map of Belarus).
